Frequently asked questions

What counts as a task run?

A task run is a single execution of a background job. If your job calls child tasks, each child task counts as a separate run.

Can I self-host Trigger.dev?

Yes, Trigger.dev is open source and can be self-hosted. The cloud version is managed for you with automatic scaling and updates.
